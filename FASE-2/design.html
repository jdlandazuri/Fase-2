Etiquetas básicas



Html es un lenguaje de etiquetas, en la cual construiremos nuestra primera pagina web, una página web se construye con etiquetas básicas como: etiqueta head, header, body, style, links, h1, etiqueta p, div, nav, section, article, aside, footer, imagenes, video, listas, ordenadas y desodenadas.

Atributos



Los atributos son las propiedades de las etiquetas, algunas de estas pripiedades se o caracteristicas son propias de la etiqueta esto quiere decir que siempre deben ir con la etiqueta, tambien algunas de estas propiedades o caráteristicas se la damos nosotros desde el código, con el fin de dar identiicación de la etiqueta o de alguna secctión.

valores



Algunas etiquetas de HTML solicitan valores, así mismo podemos mostrar esos valores, a estos valores solicitados se les llaman entradas, y los valores mostrados se les llaman valores de salida.
